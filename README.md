# Fit Freak – Smart Diet & Meal Planner

## Project Overview

Fit Freak is a web-based application that helps users discover healthy meals, analyze nutritional values, and generate personalized diet plans. The application uses the Spoonacular API to fetch real-time recipe data, including images, nutrition, and meal planning.

---

## Features

### Food Images

* Displays high-quality images for each recipe
* Enhances user experience with visual meal browsing

---

### Nutrition Data

* Shows detailed nutritional information:

  * Calories
  * Protein
  * Carbohydrates
  * Fat
* Helps users make informed dietary decisions

---

### Meal Planning

* Generates:

  * Daily meal plans
  * Weekly meal plans
* Based on:

  * Calorie requirements
  * Diet type (vegetarian, keto, etc.)

---

### Ingredients & Quantity

* Displays:

  * Ingredients list
  * Exact quantity required for each recipe

---

### Search Functionality

* Search recipes by:

  * Name
  * Ingredients
  * Diet type

## How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/fit-freak-meal-planner.git
   ```

2. Open `index.html` in your browser

3. Add your Spoonacular API key inside `script.js`
